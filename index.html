Data Mining — Analyse et prédiction criminologique Machine Learning

Analyse d'une base de 638 454 homicides aux États-Unis (Murder Accountability Project) pour mieux comprendre les profils des victimes et des auteurs, puis prédiction de la relation victime-auteur dans les affaires non résolues à l'aide de modèles de machine learning.

PythonAnalyse de donnéesClusteringMachine Learning
Acquis & compétences Nettoyage et préparation d'un très gros volume de données, tests statistiques pour identifier les liens entre variables, regroupement par clustering, et comparaison de plusieurs modèles de prédiction.
Difficultés rencontrées Avec un dataset aussi volumineux, nos premiers tests statistiques classiques ne donnaient plus de résultats exploitables. Nous avons dû changer de méthode et tester plusieurs façons de regrouper les données avant de trouver la combinaison la plus fiable.

Automatisation d'une migration SQL — NoSQL Bases de données

Conception d'une base de données structurée, puis développement d'un système automatisé pour migrer les données d'une base relationnelle vers une base NoSQL, en garantissant la cohérence des informations.

SQLPythonBases de données
Acquis & compétences Modélisation de données, abstraction orientée objet, migration de données, Python, SQLAlchemy.
Difficultés rencontrées La principale difficulté a été de faire correspondre la structure d'une base relationnelle (organisée en tables liées) à une base NoSQL (organisée en documents). Il a fallu repenser l'organisation des données pour ne perdre aucune information lors du passage d'un format à l'autre.
Capture / image du projet
SAE Amazon — Prédiction de notes à partir d'avis clients Machine Learning · NLP

Challenge en équipe basé sur le dataset Amazon Reviews 2023 : à partir du titre, du texte d'un avis et d'informations sur le produit, l'objectif était de prédire automatiquement la note laissée par le client (de 1 à 5). Les résultats étaient évalués selon l'écart moyen entre la note prédite et la note réelle (plus l'écart est petit, meilleur est le modèle).

PythonTraitement du texteMachine Learning
Acquis & compétences Nettoyage de données textuelles, transformation du texte en données exploitables par un modèle, comparaison méthodique de plusieurs approches pour choisir la plus performante face à un objectif de régression.
Difficultés rencontrées Le texte des avis ne pouvait pas être utilisé directement par les modèles : il a fallu tester plusieurs façons de le transformer en données numériques, certaines étant beaucoup plus longues à calculer que d'autres, pour trouver le bon compromis entre qualité des résultats et temps de traitement.
